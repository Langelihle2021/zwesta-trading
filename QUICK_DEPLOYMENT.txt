## ⚠️ COMMON ISSUES & FIXES

### "Port 80 already in use"
```powershell
# Find what's using port 80
netstat -ano | findstr :80

# Kill the process by PID
taskkill /PID [PID] /F

# Or use port 8080 instead
python -m http.server 8080
# Then access: http://38.247.146.198:8080
```

### "Logo not displaying"
1. Hard refresh browser: **Ctrl+Shift+R**
2. Clear cache: **Ctrl+Shift+Del**
3. Check file exists: `C:\zwesta-trader-web\assets\images\logo.png`
4. Try different browser

### "Cannot GET / error"
1. Verify server running: `Get-Process python`
2. Verify correct folder: `cd C:\zwesta-trader-web && ls`
3. Check index.html exists: `ls index.html`
4. Wait 5 seconds and refresh page

### "Service Worker Failed" errors
These are normal! Just:
1. Refresh page
2. Or use incognito mode
3. Or clear browser cache

**App works fine despite these errors.**
